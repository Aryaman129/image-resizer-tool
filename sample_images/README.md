# Sample Images Directory

This folder should contain your input images for resizing.

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- BMP (.bmp)
- GIF (.gif)
- TIFF (.tiff)
- WebP (.webp)

## Usage

Place your images in this folder and run the resizer script to process them.

Example:
```bash
python image_resizer.py -i sample_images -o output -w 800
```

This will resize all images in this folder to 800px width while maintaining aspect ratio.
